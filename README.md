# 🧪 TestGen AI
**Transforming requirements into quality assurance in seconds.**

TestGen AI is an intelligent QA engine that bridges the gap between product descriptions and execution. By leveraging LLMs (via Groq API), it converts plain English feature specs into high-quality manual test cases and production-ready automation scripts instantly.

---

### 🚀 Key Features

* **📋 Instant Manual Cases:** Generate structured, step-by-step test cases from simple text.
* **⚙️ Multi-Framework Automation:** Export scripts for **Selenium, Cypress, or Playwright** on the fly.
* **📝 Script-to-Script Mode:** Refine or translate existing scripts using AI.
* **🗂 Project Hub:** Organize your testing suites by project and track generation history.
* **💬 AI QA Assistant:** Integrated chat for real-time debugging and strategy brainstorming.
* **🔐 Enterprise-Grade Auth:** Secure access via JWT-based authentication.
* **👤 Instant Demo:** Explore the full feature set via a "no-signup" guest login.

| Layer | Technologies |

---

### 📦 System Requirements

To ensure a smooth setup, please verify your environment matches these versions:

* **Java:** 17 (Strictly required; newer versions may conflict with Lombok)
* **Maven:** 3.9+
* **PostgreSQL:** 16
* **Node.js:** 18+
* **API Access:** A valid **API Key**

---

> [!IMPORTANT]  
> **Development Note:** Ensure your IDE is configured for **Java 17**. Using Java 21+ may result in compilation errors regarding annotation processing and Lombok.
